# Web Keybinder

Browser extension for creating custom keyboard shortcuts on web sites to help productivity and navigation

## Features

- Create shortcuts and assign elements such as, buttons, input fields, and links
- Autoload the keybinds for specific site
- Create, delete, and manage your shortcuts through a clean UI

## Installation (for Chrome)

1. Clone the git repository

```bash
git clone https://github.com/mashrursakif/web-keybinder.git
```

2. Open Chrome and click manage extensions (or visit chrome://extensions/)
3. Click `Load Unpacked` and select the repository

The extension is now ready to use!

## Usage

1. Click the extension to open the popup.
2. View and manage all keybinds and click `Add New` to create new ones
3. Record a new keybind and select the element to assign it to.
4. Added shortcuts will be autoloaded and synced
